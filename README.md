# 🏧 ATM Simulator

A web-based ATM Simulator built using **HTML, CSS, and JavaScript** that replicates core ATM functionalities such as PIN authentication, balance inquiry, cash withdrawal, cash deposit, transaction history, and multilingual support.

## 🚀 Features

* Secure PIN Validation
* Multiple User Support
* Hindi & English Language Selection
* Savings and Current Account Selection
* Balance Inquiry
* Cash Deposit
* Cash Withdrawal
* Daily Withdrawal Limit
* Account Lock after Multiple Wrong PIN Attempts
* Transaction History Tracking
* Interactive ATM-like User Interface

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## 📋 How It Works

1. Insert Card (Enter Username)
2. Select Language
3. Select Account Type
4. Enter PIN
5. Access ATM Services:

   * Check Balance
   * Deposit Money
   * Withdraw Money
   * View Transaction History
   * Exit ATM

## 🔒 Security Features

* PIN Authentication
* Limited Login Attempts
* Temporary Account Lock on Multiple Failed Attempts
* Daily Withdrawal Limit Validation

## 💡 Project Purpose

This project was built to understand how ATM systems handle:

* User Authentication
* Transaction Processing
* Security Validation
* Account Management
* User Interaction Flow

It also helped strengthen my JavaScript problem-solving skills and frontend development concepts.
